# New-Base-GM-Mini-App-With-Open-Source-Codes
We propose building an open-source “BasedGM” mini-app on Coinbase’s Base layer-2 that lets users send a daily on-chain “GM” transaction.
Users connect a wallet (e.g. Rabby or MetaMask via Ethers.js or Wagmi) and click a GM button each day. A simple smart contract records each call, tracking daily streaks and totals. Milestone achievements (e.g. 7-day streak) can mint a soulbound NFT badge via ERC‑5192. We balance on-chain vs off-chain storage: core data (timestamps, streak counts) can be stored on-chain for transparency, while leaderboards or referral aggregates can be computed off-chain from contract events. Base’s low gas fees (≈0.005 Gwei per tx) make daily calls nearly free, enabling a smooth user experience. The frontend (React) uses Wagmi/Ethers hooks to connect wallets and call the contract, and the site can be deployed on Vercel or Netlify with CI/CD pipelines. We include full annotated Solidity contracts, React code, deployment scripts (Hardhat), testing strategy, CI setup, and file structure. Our design mirrors features of basedgm.fun (daily “GM”/“BM” clicks, streaks, referrals, leaderboards, badges) in a secure, gas-optimized way on Base.

Background and Features
The original BASED GM app (basedgm.fun) is a Base-native daily on-chain ritual with GM/BM buttons, streak counters, referrals, leaderboards, and soulbound NFT badges. (E.g. OnChainGM is a multi-chain analog enabling daily GM presence.) Our mini-app will replicate core flows on Base: user clicks GM (or BM) once per day, sending a transaction to our GM contract. The contract enforces one call per ~24 h, updates the user’s streak and total, and emits an event. A referral link feature can record who referred a new user, rewarding both. A leaderboard (off-chain) ranks addresses by total GMs or streaks. When users hit milestones (e.g. 7, 30, 100 days), the contract (or a connected owner) can mint a non-transferable badge NFT (ERC-5192) to them. This gamifies on-chain activity and builds community.
